# 菜单
## 菜单属性定义
1. 菜单ID格式定义为`aa.bb.cc`(如100100)
  1. aa=10，代表一个模块
  2. bb=01，代表这个模块下的一个菜单
  3. cc=00，代表这个菜单下的功能列表

## 菜单模块表格
| 模块     | 名称    | ID       |
|:--------|:--------|:---------|
| fa-base | 基础模块  | 0-100000 |
| fa-demo | 演示模块  | 100000   |
| fa-app  | APP模块 | 200000   |
| fa-disk | 网盘模块  | 300000   |
